1. Gdzie wstawiany jest styl wewnętrzny?

Styl wewnętrzny wstawiany jest w sekcji head dokumentu HTML, wewnątrz znacznika <style>.

2. Składnia teoretyczna (w ramce, wcięcia):

<style>
selektor {
  cecha: wartość;
}
</style>

3. Jakie mogą być selektory?

Selektorami mogą być znaczniki HTML (np. p, h1), klasy (.klasa), identyfikatory (#id) oraz inne bardziej zaawansowane selektory.

4. Jak wpisujemy cechy i wartości?

Cechy wpisujemy jako właściwości CSS, np. color, font-size. Po dwukropku wpisujemy wartość, np. red, 20px. Każdą deklarację kończymy średnikiem.
