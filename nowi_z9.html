1. Składnia teoretyczna (ramka):

<link rel="stylesheet" href="style.css">

2. Gdzie wstawiany jest styl zewnętrzny?

Styl zewnętrzny wstawiany jest w sekcji head dokumentu HTML, za pomocą znacznika <link>.

3. Najprostsza forma wstawienia stylu:

<link rel="stylesheet" href="style.css">

4. Ile arkuszy można podłączyć?

Do jednego dokumentu HTML można podłączyć wiele arkuszy stylów CSS.

5. Co się dzieje przy konflikcie stylów?

W przypadku konfliktu ważniejszy jest styl zapisany później lub bardziej szczegółowy.

6. Jak tworzymy plik CSS?

Plik CSS można utworzyć w zwykłym edytorze tekstu, np. Notatniku, zapisując go z rozszerzeniem .css.

7. Jakie są edytory CSS?

Można używać specjalnych edytorów kodu, np. Visual Studio Code, Notepad++.

8. Jakie rozszerzenie ma plik CSS?

Plik CSS posiada rozszerzenie .css.

9. Zalety stylu zewnętrznego:

  • łatwe zarządzanie wyglądem wielu stron
  • oddzielenie treści od stylu
  • mniejszy kod HTML

10. Uwagi na temat stylów:

CSS jest wrażliwy na błędy składniowe, dlatego należy dokładnie sprawdzać kod.
